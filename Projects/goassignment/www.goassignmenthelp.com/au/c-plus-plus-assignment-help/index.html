C++ Programming

~ Important Concepts to Know before Writing C++ Assignments ~ 

C++ was developed in 1979 and for the past few years, there is a large focus on improving C++. It is one of the most popular programming languages ever created. 

C++ is a general-purpose programming language that is used to write software in a wide variety of application domains such as system software, application software, device drivers, embedded software, servers, client applications and entertainment software. It is also used for hardware design and it is the only language which is built around scope-bound Resource Management, SBRM concept (also known as RAII).

C++ encapsulates the features of both low and high-level programming language. It is a portable, structured, compiler-based, object-oriented language with a rich standard library.
